# largemodel
1. 环境要求
* Python 3.9
* conda

3. 克隆仓库
    * git clone https://github.com/1111222234/large_model.git
    * cd large_model

3. 创建并激活虚拟环境
    * conda create -n env python==3.9
    * conda activavre env
4. 安装依赖
所有必需的库都已列在 requirements.txt 文件中。
    * pip install -r requirements.txt

6. 下载数据集
数据集，及脚本已下载好，直接使用即可

7. 运行
    * python main.py
